# Habit Tracker

A configurable personal habit tracker that runs as a local HTML file and can be installed as a PWA on iOS.

## Setup

1. Download `dist/index.html` from [Releases](../../releases)
2. Open in Safari on iOS
3. Tap Share → Add to Home Screen

## Development

```bash
npm install
npm test        # run all tests
npm run dev     # local dev server
npm run build   # produce dist/index.html
```

## Data

All data is stored locally on your device. Use the Export button to back up your data as JSON, and Import to restore it.
